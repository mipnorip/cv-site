# Сайт-визитка Ильи Маслова

Персональный сайт-визитка (портфолио) для Ильи Маслова, DevOps инженера.

## Технологии

- **TypeScript** - типизированный JavaScript
- **Vite** - быстрая сборка и разработка
- **CSS Modules** - изолированные стили
- **Vanilla JS** - без фреймворков

## Структура проекта

```
.
├── index.html          # Главная HTML страница (в корне проекта)
├── src/
│   ├── main.ts         # Точка входа приложения
│   ├── types/          # TypeScript типы
│   ├── data/           # JSON данные (личные данные, опыт, проекты, changelog)
│   ├── components/     # Компоненты приложения
│   ├── styles/         # CSS Modules стили
│   └── utils/          # Утилиты
```

## Установка и запуск

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

Сайт будет доступен по адресу `http://localhost:5173`

### Сборка для production

```bash
npm run build
```

Собранные файлы будут в директории `dist/`

### Просмотр production сборки

```bash
npm run preview
```

## Настройка данных

Все данные хранятся в JSON файлах в директории `src/data/`:

- `personal.json` - личные данные (имя, возраст, контакты, фото)
- `experience.json` - опыт работы
- `projects.json` - личные проекты и увлечения
- `changelog.json` - еженедельный changelog

Отредактируйте эти файлы, чтобы обновить информацию на сайте.

**Примечание:** Добавьте ваше фото в `public/assets/images/photo.jpg` или обновите путь к фото в `personal.json`. Файлы в папке `public/` доступны по корневому пути (например, `public/assets/images/photo.jpg` → `/assets/images/photo.jpg`).

## Разделы сайта

1. **О себе** - личные данные, фото, контакты
2. **Опыт** - хронология работы с описанием задач и технологий
3. **Проекты** - личные проекты и увлечения с фильтрацией
4. **Changelog** - подробный отчет о задачах, решенных на каждой неделе

## Особенности

- Адаптивный дизайн (mobile-first)
- Плавная прокрутка к секциям
- Интерактивный changelog с разворачиванием/сворачиванием недель
- Фильтрация проектов по типу
- Мобильное меню навигации
